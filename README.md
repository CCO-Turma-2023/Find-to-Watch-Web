# Find-to-Watch-Web

Este projeto é uma aplicação web desenvolvida para ajudar os usuários a descobrir e organizar filmes e séries para assistir. Ele oferece uma interface intuitiva para pesquisar títulos, ver detalhes e gerenciar uma lista personalizada de conteúdo.

## Funcionalidades

- **Pesquisa de Títulos:** Encontre filmes e séries por nome.
- **Detalhes do Conteúdo:** Visualize informações como sinopse, elenco, avaliações e trailers.
- **Lista de Observação:** Adicione e gerencie filmes e séries que você deseja assistir.

## Como Usar

1.  **Clonar o Repositório:**
    ```bash
    git clone https://github.com/seu-usuario/Find-to-Watch-Web.git
    cd Find-to-Watch-Web
    ```
2.  **Instalar Dependências:**
    ```bash
    npm install # ou yarn install
    ```
3.  **Configurar Variáveis de Ambiente:**
    Crie um arquivo `.env` na raiz do projeto e adicione as chaves de API necessárias (ex: para TMDB).
    ```
    REACT_APP_TMDB_API_KEY=sua_chave_aqui
    ```
4.  **Iniciar a Aplicação:**
    ```bash
    npm start # ou yarn start
    ```
    A aplicação estará disponível em `http://localhost:5173`.

## Tecnologias Utilizadas

- **Frontend:** React, HTML, CSS, JavaScript
- **API:** The Movie Database (TMDB) API
- **Estilização:** Tailwind CSS
